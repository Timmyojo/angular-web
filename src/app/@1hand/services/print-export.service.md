# PrintExportService - Service d'impression et d'export

Ce service centralisé permet d'imprimer et d'exporter des listes de données de manière réutilisable dans toute l'application.

## Fonctionnalités

- **Impression** : Génère un document HTML formaté et l'ouvre dans une nouvelle fenêtre pour impression
- **Export CSV** : Exporte les données au format CSV avec téléchargement automatique
- **Configuration flexible** : Colonnes personnalisables avec formatage, alignement et largeur
- **Informations d'en-tête** : Affichage d'informations contextuelles
- **Styles personnalisés** : Possibilité d'ajouter des styles CSS personnalisés
